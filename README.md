# Etsy-test-suite
This repository holds all automated test cases for the Etsy features!

## Why?
This project is to display testing knowledge and attempt mostly automated UI test suites. 

## Dependencies 
You will need Playwright installed for the tests to show up and work.

npm init playwright@latest 
or 
install the VS Code Playwright extension and input '>Test: Install Playwright' command in the search bar.
